Game of Thrones API

Instructions

Select all

/api/characters?limit=X

Input the limit as a number instead of X
If no limit is provided all characters will be shown.


Search

/api/characters/[character_name]

You need to input full name separated with "_".


Add

/api/characters?name=new_name

You need to send an object with all the necessary fields.
It can be only tested with tools like Insomnia or Postman.

{
"characterName": "Mia",
"houseName": "Targaryen",
"characterImageThumb": "https://imageThumb.jpg",
"characterImageFull": "https://imageFull.jpg",
"actorName": null,
"killed": ["Dragon"],
"killedBy": null
}


Update

/api/characters/[character_name]?name=new_name

At the moment you can only edit names.
It can be only tested with tools like Insomnia or Postman.


Delete

/api/characters/[character_name]

It can be only tested with tools like Insomnia or Postman.
